XDC-http-cache

.. contents:: Table of Contents

**XDC HTTP Cache** - Distributed system of HTTP-based storage

This project includes a playground for a demonstrator of a distributed system of storage, comprising 
storage elements, caches and redirectors, communicating via HTTPS.
